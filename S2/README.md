# ⚙️ Boilerplate Code & Variables

## 📌 Overview

This module introduces the basic structure of a program (boilerplate code) and the concept of variables, which are essential building blocks in programming.

---

## 🧱 What is Boilerplate Code?

Boilerplate code refers to the standard structure or template of a program that is required before writing the actual logic.

👉 It is the **starting point of every program**

---

## 💻 Example (C++ Boilerplate Code):

```cpp
#include <iostream>   // Header file

using namespace std;  // Standard namespace

int main() {          // Main function (entry point)

    // Your code starts here

    return 0;         // End of program
}
```

---

## 🔍 Understanding the Boilerplate

* `#include <iostream>` → Used for input/output
* `using namespace std` → Avoids writing std:: again and again
* `int main()` → Starting point of program execution
* `return 0` → Indicates successful execution

---

## 📦 What are Variables?

Variables are containers used to store data values.

👉 In simple words:
**"Variable = A box that stores data"**

---

## 🧾 Types of Variables in C++

| Type   | Description       | Example      |
| ------ | ----------------- | ------------ |
| int    | Integer values    | 10, -5       |
| float  | Decimal numbers   | 3.14         |
| char   | Single character  | 'A'          |
| string | Text data         | "Hello"      |
| bool   | True/False values | true / false |

---

## 💻 Example of Variables:

```cpp
#include <iostream>
using namespace std;

int main() {
    int age = 20;
    float marks = 85.5;
    char grade = 'A';
    string name = "Shiv";
    bool isStudent = true;

    cout << name << " is " << age << " years old.";
    
    return 0;
}
```

---

## ⚙️ Rules for Naming Variables

* Must start with a letter or underscore
* Cannot start with a number
* No spaces allowed
* Cannot use keywords (like int, float)

---

## 🚀 Why Variables are Important?

* Store data for processing
* Make programs dynamic
* Help in calculations and logic building
