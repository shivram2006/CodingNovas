# 💻 Introduction to Programming, Code & IDE

## 📌 Overview

This module is designed to introduce beginners to the world of programming. It covers the fundamental concepts of what programming is, how code works, and how to use an IDE (Integrated Development Environment).

---

## 🧠 What is Programming?

Programming is the process of giving instructions to a computer to perform specific tasks.

👉 In simple words:
**"Programming = Talking to a computer in its language"**

---

## 🧾 What is Code?

Code is a set of instructions written in a programming language (like C++, Python, Java) that a computer can understand and execute.

### Example (C++ Code):

```cpp
#include <iostream>
using namespace std;

int main() {
    cout << "Hello, CodingNovas!";
    return 0;
}
```

👉 Output:

```
Hello, CodingNovas!
```

---

## ⚙️ What is an IDE?

IDE (Integrated Development Environment) is a software tool that helps programmers write, run, and debug code easily.

### 🔧 Popular IDEs:

* VS Code
* Code::Blocks
* PyCharm
* IntelliJ IDEA

---

## 🚀 Why Use an IDE?

* Easy code writing
* Syntax highlighting
* Error detection
* One-click run & debug

---

## 🛠️ Basic Steps to Run a Program

1. Open IDE
2. Create a new file
3. Write code
4. Save the file
5. Click Run ▶️

---

## 🎯 Learning Outcome

After completing this module, you will:

* Understand what programming is
* Know how code works
* Be able to use an IDE
* Run your first program successfully

---

## 💡 Pro Tip

Start simple. Focus on logic, not just syntax.
